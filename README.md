# EpinelPS-EZButton
EpinelPS一键端，无需分布启动

功能列表：

在线下载：下载EpinelPS作者仓库最新压缩包、NIKKE旧版本安装包

检测网络：检测是否PC是否处于TUN模式网络环境。

一键启动：三个软件分步进行，配置好环境后下一步

自动查找：全盘查找对应文件，仅限EpinelPS.exe、ServerSelector.Desktop.exe、nikke_launcher.exe。

小功能：

桌面文件夹：用于存放EpinelPS作者仓库最新压缩包、NIKKE旧版本安装包，易寻找。

检测账号：PC账户是否处于微软账号并生成到相应桌面（微软账号登录后会出现:驱动盘\OneDrive\Desktop，而未登录的桌面为：驱动盘\Desktop）
